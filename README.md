# Students' Social Media Addiction

## 📌 Project Overview
This project analyzes patterns of social media usage and addiction among students aged **16–25** using the *Student Social Media & Relationships* dataset. The study explores how usage intensity relates to mental health, sleep behavior, and interpersonal relationship outcomes.

The analysis combines **data preparation, exploratory data analysis (EDA), correlation studies, segmentation**, and **interactive visualizations** developed in **Power BI**.

---

## 🎯 Objectives
The project aims to:

- Examine relationships between:
  - **Average Daily Usage Hours** and **Mental Health Score**
  - **Average Daily Usage Hours** and **Sleep Hours per Night**

- Analyze **relationship conflicts** based on:
  - Usage intensity
  - Platform preferences

- Identify **student user segments**, such as:
  - High-usage / high-stress
  - Moderate-usage / balanced
